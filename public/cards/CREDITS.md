# Card art credits

The 52 card-face images and `back.png` in this folder come from the
[`playing-cards-assets`](https://github.com/hayeah/playing-cards-assets)
repository (MIT licensed packaging) by Howard Yeh, which is itself derived from
the **public-domain** "Vectorized Playing Cards" set
(originally https://code.google.com/p/vector-playing-cards/, released to the
public domain).

- Card faces: **public domain**.
- Packaging/processing: MIT License — see the source repository.

These are safe to redistribute in this game, including the Play Store build.
